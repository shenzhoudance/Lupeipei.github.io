如何为你的blog添加文章阅读量统计？

发表于 2017-11-14 | 分类于 Hexo | | 阅读次数

写在前面

这两天给自己的blog加上了阅读量统计，记录下整个的实现过程，给需要的小伙伴一个参考。

以下过程是针对NexT主题的blog，使用了第三方服务leancloud。

正文

一、配置leancloud

  • 进入leancloud主页，完成登陆注册。

  • 创建一个应用，应用的名字可以随喜好来定，后面也可以在设置中进行修改。

    这里我创建了一个名为counter的应用，

  • 创建好应用后，选择 存储 - 创建 class， 写入class的名称，这里为了保证对NexT主题的修改兼容，请一定命名为Counter，权限建议选择无限制，或者限制写入也OK的。

    【你可以把Counter看做是用来存储你的文章浏览情况的，呃，貌似它就是这样的。】

    ​

  • 选择设置 - 应用Key, 我们拿到App ID ,App Key, 这两个关键参数是要写入你NexT主题的配置文件的。【吐糟下，这两个参数目前（2017.11）是不能复制的，只能手敲进config文件。】

二、设置NexT主题

  • 在NexT主题的_config.yml文件中，加入App ID和key。

  • 终端输入,重新部署hexo ：

    应该就可以正常使用阅读量统计的功能了。

    这时候，文章的初始阅读量显示为0，点击后会生成记录，这些数据会记录在先前创建的Counter表中。

三、安全性

​ 因为App ID和Key都是暴露在外的，为了保证安全性，建议开启Web安全。选择设置 - 安全中心，填写上你的blog域名。

OK，大功告成。

Enjoy : P

The End
